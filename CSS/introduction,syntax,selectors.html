CSS Introduction

CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of a webpage written in HTML or XML. CSS is used for describing the look and formatting of a webpage, including the layout, colors, and fonts.

CSS Syntax

The syntax of CSS is made up of selectors, properties, and values. Selectors are used to select the element(s) you want to style, properties are the attributes of the element(s) you want to style, and values are the values you want to assign to the properties.

For example, the following CSS code selects all the h1 elements on a webpage and assigns them the color red:

h1 { color: red; }

CSS Selectors

CSS selectors are used to select the element(s) you want to style. There are many different types of selectors, including:

  • The universal selector (*) which selects all elements
  • The type selector which selects an element based on its type
  • The class selector which selects an element based on its class name
  • The id selector which selects an element based on its id

For example, the following CSS code selects all the elements with the class name "red" and assigns them the color red:

.red { color: red; }

For example, the following CSS code selects the element with the id "blue" and assigns it the color blue:

#blue { color: blue; }
