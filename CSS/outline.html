CSS Outline

The CSS outline property is used to add an outline around an element. The outline is different from the border in that it is not a part of the element's width or height.

Outline types:

  • None: No outline
  • Dotted: Dotted outline
  • Dashed: Dashed outline
  • Solid: Solid outline
  • Double: Double outline
  • Groove: Grooved outline
  • Ridge: Ridged outline
  • Inset: Inset outline
  • Outset: Outset outline
